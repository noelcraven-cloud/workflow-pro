# Workflow Pro MVP

## Product Vision

Workflow Pro is a workflow orchestration tool for finance teams.

Its purpose is to answer:

1. What is the next most important task?
2. Who owns it?
3. When should it be done?

---

## Core Principles

- Everything starts in Triage.
- Triage is temporary.
- Tasks leave Triage automatically once fully triaged.
- One task exists once.
- Views are different ways of looking at the same task.
- Ranked work and BAU work are different workflow types.

## Workflow Types

### Ranked

Each assigned person has an independent ranked list.

Ranks:

- always begin at 1
- have no gaps
- automatically renumber

### Weekly BAU

Recurring weekdays.

Multiple weekdays allowed.

### Monthly BAU

Recurring financial working days.

Supported cycle:

WD-3

WD-2

WD-1

WD+1

...

WD+22
